# Croom

**Croom** is a realtime chat web application. The main feature of the app is that the users can start a study session on thier respective chat room. When users started a study session they will be given a form to fill up questions and answers, after filling up the question and answers the users will be given a random question picked from the form; a user cannot be asked by the question they wrote. After the user finishes the quiz, thier scores will be displayed and thier questions with thier answer will be mereged and displayed and has the option to donwload it as a PDF.
