# Croom

**Croom** is a realtime chat web application. The main feature of the app is that the users can start a study session on their respective chat room. When users started a study session they will be given a form to fill up questions and answers, after filling up the question and answers the users will be given a random question picked from the form; a user cannot be asked by the question they wrote. After the user finishes the quiz, their scores will be displayed and their questions with their answer will be merged and displayed and has the option to download it as a PDF.
